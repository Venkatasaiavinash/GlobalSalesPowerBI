# GlobalSalesPowerBI
Interactive Power BI dashboard analyzing global sales performance using DAX, Power Query, and data visualization techniques to deliver business insights.

The Global Sales Performance Dashboard is an interactive Business Intelligence (BI) project built using Microsoft Power BI. It provides valuable insights into global sales data by analyzing revenue, profit, customer behavior, product performance, and regional trends through dynamic visualizations.

**Features**
 Interactive Sales Dashboard
 Region-wise Sales Analysis
 Profit & Revenue Tracking
 Product Performance Analysis
 Customer Insights
 Year-to-Date (YTD) & Previous Year (PY) Comparison
 KPI Cards for Quick Business Overview
 Dynamic Filters and Slicers
 Trend Analysis using Charts and Maps
